# 国标麻将助手

Android 端国标麻将辅助工具，支持手牌分析、听牌计算、番种判定和剩余牌数统计。

## 功能

- **手牌输入**：34 种牌型网格点选，支持手牌和牌河分别记录
- **听牌计算**：根据当前手牌自动计算听哪些牌（支持标准形、七对、十三幺等所有听形）
- **番种判定**：约 40 种国标番种自动检测（88番大四喜/大三元/绿一色 到 1番自摸/花牌）
- **剩余张数**：结合场上牌河统计每张听牌的实际剩余枚数
- **发展分析**：替换式分析（弃N摸N），按番种聚合发展概率，点击查看改进牌明细
- **番数自定义**：番数规则页支持任意番种的数值调整和隐藏，适配不同规则版本
- **分析规则**：可调节替换深度（1~3），设置自动持久化

## 构建

```bash

./gradlew :app:installDebug
```

## 算法

- **和牌分解**：DFS 回溯 + 最低位优先策略，枚举全部分解
- **听牌计算**：对 34 种牌型逐一试加后调用和牌判定（天然覆盖九莲宝灯 9 面听、十三幺 13 面听等）
- **番种计分**：逐条 detector → 按不计关系排除被含低番 → 累加 → 判 8 番起和
- **剩余张数**：每种牌 4 − (自暗手 + 自副露 + 全家牌河 + 他家副露)

## 已知限制

- 全不靠/七星不靠规则定义待官方核实，暂未启用
- 番种已实现约 47 种（含豪华七对系列/红孔雀/蓝一色/大七星），低频番种待补全
- 相机自动识别为二期规划
- 番数覆盖暂存内存，未做持久化
