# 雀门助手

Android 端国标麻将辅助工具，支持手牌分析、听牌计算、番种判定和剩余牌数统计。

## 功能

- **手牌输入**：34 种牌型网格点选，支持手牌和牌河分别记录，剩余张数角标展示
- **听牌计算**：根据当前手牌自动计算听哪些牌（标准形、七对、豪华七对、十三幺等）
- **番种判定**：21 种精番种自动检测（混一色/碰碰胡 3番 到 大四喜/九莲宝灯/大七星 24番）
- **剩余张数**：结合场上牌河统计每张听牌的实际剩余枚数
- **发展分析**：番种倒推式替换分析（弃N摸N），快速向听预检 + 渐进深度，深度可调（1~3）
- **番数自定义**：番数规则页支持番种数值调整和隐藏，适配不同规则版本
- **分析规则**：可调节替换深度，设置自动持久化

## 构建

```bash
# 运行引擎测试（无需设备）
./gradlew :engine:test

# 构建调试版 APK
./gradlew :app:assembleDebug
# 输出: app/build/outputs/apk/debug/app-debug.apk

# 安装到已连接设备
./gradlew :app:installDebug
```

### 环境要求

- Android 8.0+ (API 26)
- JDK 17、Gradle 8.11.1、AGP 8.7.2
- Kotlin 1.9.22、Jetpack Compose + Material3
- Android SDK（在 `local.properties` 中配置 `sdk.dir`）

## 算法

- **和牌分解**：DFS 回溯 + 最低位优先策略，枚举全部分解
- **听牌计算**：34 路枚举 × 和牌判定（覆盖九莲宝灯 9 面听、十三幺 13 面听等）
- **番种计分**：逐条 detector → 按不计关系排除被含低番 → 累加
- **发展分析**：番种倒推（不合规牌统计 → 目标牌池枚举 → 快速向听预检 → 全量验证），搜索量比通用枚举低 100~10000 倍
- **剩余张数**：每种牌 4 − (自暗手 + 自副露 + 全家牌河 + 他家副露)

## 已知限制
